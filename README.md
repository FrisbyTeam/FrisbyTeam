## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your bot token:
   ```
   cp .env.example .env
   ```

3. Paste your code into `bot.py` and run:
   ```
   python bot.py
   ```
